# USC Economics AI Working Group

Welcome to the resource hub for the AI Working Group in the Department of Economics at the University of South Carolina.

This site collects guides, tutorials, and curated materials to help department members integrate AI tools into their research workflows — covering everything from writing and literature review to coding and data analysis.

## How to Use This Resource

The materials are organized into chapters by topic. If you are new to the tools discussed here, start with the **Background Tools** chapter, which covers foundational software skills that will make working with AI tools much smoother.

## Contributing

This is a living resource. If you have materials, workflows, or tips to share, contributions via pull request to the [GitHub repository](https://github.com/jdebacker/AI-Working-Group) are welcome.
