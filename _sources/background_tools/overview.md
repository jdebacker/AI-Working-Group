# Background Tools

Before diving into AI-specific workflows, it helps to have a solid foundation in a few general-purpose tools that are widely used in research and software development. The pages in this chapter cover:

- **Git and GitHub** — version control and collaborative project management
- **Markdown** — a lightweight syntax for writing formatted documents (including this one)
- **Text Editors** — choosing and configuring an editor that supports your workflow

These tools are not AI-specific, but they come up constantly in AI-assisted research contexts. Many AI tools generate markdown output, operate inside editors, and are best used alongside a version-controlled project structure.
